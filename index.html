ℹ️ About Fluid Sorting Simulator

🎯 Purpose

Train your sorting skills with progressive difficulty levels designed to improve speed and accuracy in fluid sorting operations.

🎮 How to Play

  • Watch items move on the conveyor belt
  • Press SPACE or click PICK when items are in the pick zone
  • Use keys 1-4 or click buttons to drop items to correct nodes
  • Ignore items that don't belong to any node
  • Complete all 40 items to finish the level

🔧 Hardware Support

Connect an Arduino device via Web Serial API for physical controls. Requires Chrome or Edge browser with HTTPS connection.

📊 Features

  • 4 difficulty levels from Beginner to Expert
  • Real-time performance tracking
  • Detailed game history and analytics
  • Keyboard and hardware controls
  • Export training data to CSV

⌨️ Keyboard Shortcuts

Space Pick Item
1-4 Drop to Node
Ctrl+P Training Tab
Ctrl+H History Tab
Ctrl+1-4 Start Level
Ctrl+R Random Level

🏆 Scoring System

✅ Correct: Item picked and dropped to correct node
❌ Missorted: Item picked but dropped to wrong node
⚠️ Spillover: Relevant item not picked in time
🚫 False Pick: Irrelevant item picked unnecessarily
👍 Ignored: Irrelevant item correctly ignored

🔗 Technical Info

Version: 2.0.0
Browser Support: Chrome 89+, Edge 89+, Firefox 87+
Hardware API: Web Serial API
Storage: Local Storage
Framework: Vanilla JavaScript ES6+
